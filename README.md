# Password Strength Checker

This project is a Python program that evaluates the strength of a password.

## Features
- Checks for:
  - Length
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Gives feedback to improve weak passwords
- Loops until a strong password is entered

## How to run
1. Run the Python file
2. Enter a password
3. Follow the suggestions to improve it

## Language
- Python
